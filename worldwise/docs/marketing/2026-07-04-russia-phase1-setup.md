## 1. VK — сообщество + токен (≈15 мин)

1. vk.com → Сообщества → Создать → «Бизнес» → категория «Недвижимость». Имя/слаг/описание — выше.
2. В описании и в блоке «Ссылки» — TG-канал и WhatsApp. Первый пост можно не писать: придёт с автопостом.
3. Токен: Управление → Настройки → Работа с API → **Ключи доступа** → «Создать ключ» с правами
   **«Управление сообществом» + «Фотографии» + «Стена»**. Скопировать ключ.
4. `VK_GROUP_ID` — цифровой ID сообщества (Управление → Настройки, или vk.com/club<ID>). Положительное число.
5. На сервере: добавить в `/var/www/worldwise/.env.local`:
   ```
   VK_ACCESS_TOKEN=<ключ>
   VK_GROUP_ID=<цифры>
   ```
6. Проверка: `cd /var/www/worldwise && node --experimental-strip-types --env-file=.env.local scripts/test-social-post.mjs "Тест VK" && pm2 restart worldwise`
   (тестовый пост потом удалить со стены). Если у ключа сообщества не хватит прав на фото-аплоад —
   запасной путь: user-токен админа со scopes `wall,photos` через VK ID; скажи — сделаю инструкцию.

## 2. Одноклассники — группа + приложение (≈30 мин, самое муторное)

1. ok.ru → Группы → Создать группу → «Бизнес» → тематика «Недвижимость». Имя/описание — как выше.
2. Права разработчика: ok.ru/devaccess (заявка одобряется быстро).
3. dev.ok.ru → «Мои приложения» → создать приложение (тип External/OAuth не важен для REST) →
   получить `application_key` (публичный) и `application_secret_key`.
4. В интерфейсе приложения запросить **вечный access token** для своего аккаунта (аккаунт должен
   быть админом группы) с правами `VALUABLE_ACCESS;PHOTO_CONTENT;GROUP_CONTENT`.
5. `OK_GROUP_ID` — цифры из URL группы (ok.ru/group/<ID>).
6. В `.env.local`:
   ```
   OK_ACCESS_TOKEN=<вечный токен>
   OK_APP_KEY=<application_key>
   OK_APP_SECRET=<application_secret_key>
   OK_GROUP_ID=<цифры>
   ```
7. Проверка тем же скриптом: `... scripts/test-social-post.mjs "Тест OK"` → `pm2 restart worldwise`.

2. Стартовые требования платформы: аватар, описание, ~10 подписчиков и ≥5 публикаций — набираются
   за первую неделю-две.
3. Публикация: я готовлю статьи (расширенные версии TG-постов, 2 500–4 000 знаков) — ты вставляешь
   в Студию 2–3 раза в неделю (≈2 мин на статью). В конце каждой статьи — ссылка на TG-канал.
4. Позже (если канал пойдёт): официальный RSS-автоимпорт — потребует решения про скрытый
