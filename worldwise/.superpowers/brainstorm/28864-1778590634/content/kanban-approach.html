Kanban-доска: как добавить?

Выберите подход к интеграции Kanban в текущий CRM

A

Переключатель Table / Kanban

На странице /admin/leads появляется кнопка-тоггл. Оба вида — одни и те же данные, переключение без перезагрузки.

Плюсы

  • Всё в одном месте
  • Быстрое переключение

Минусы

  • Сложнее реализовать
B

Отдельная страница /admin/kanban

Канбан — отдельный маршрут, таблица остаётся как есть. Ссылка в навигации.

Плюсы

  • Простая реализация
  • Не трогает существующий код

Минусы

  • Два места для одних данных
C

Канбан заменяет таблицу

Таблица полностью уходит, Kanban становится основным видом лидов.

Плюсы

  • Чистый UX, один вид

Минусы

  • Теряем поиск/фильтры/CSV export
